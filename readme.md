# Web Scraper - Products

A Python project to scrape product data from public websites and export it to CSV files for analysis or business use.

## Features
- Extracts product name, price, availability, and other relevant information
- Saves results in structured CSV format
- Easy to modify for different websites
- Uses Python libraries: BeautifulSoup, Pandas, Requests

## Example Output
A sample CSV file with example output is included in the `/examples` folder.
